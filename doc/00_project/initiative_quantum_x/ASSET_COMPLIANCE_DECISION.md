# ASSET_COMPLIANCE_DECISION - quantum_x

## 资产类别优先级决策

### 决策摘要
| 决策项 | 选择 | 理由 |
|--------|------|------|
| Phase 1 资产 | 虚拟币现货/永续 | API 开放、数据免费、7x24 可测试 |
| Phase 2 资产 | 美股 ETF | 流动性好、数据丰富 |
| Phase 3 资产 | A 股/期货 | 合规要求高、需持牌 |

### 资产域详情

#### Phase 1: 虚拟币（MVP 默认）
- **交易对**: BTC/USDT, ETH/USDT, BNB/USDT
- **合约类型**: 现货 + USDT 永续
- **杠杆限制**: MVP 阶段最大 3x
- **数据源**: Binance/OKX 公开 API（免费）
- **合规范围**: 非中国大陆用户、非美国用户
- **理由**:
  - API 文档完善、接入成本低
  - 7x24 交易，测试效率高
  - 公开数据免费，无需付费订阅

#### Phase 2: 美股 ETF（后续扩展）
- **标的**: SPY, QQQ, IWM, TLT
- **数据源**: Yahoo Finance（免费）/ Polygon（付费）
- **合规范围**: 非美国居民、需经纪商账户
- **理由**:
  - 流动性极佳
  - 历史数据丰富
  - ETF 规避个股风险

#### Phase 3: A 股/期货（远期规划）
- **合规约束**: 需持牌资管/自营牌照
- **数据源**: Wind/东财（付费）
- **暂不纳入 MVP 范围**

## 合规范围定义

### 用户地理限制
| 地区 | MVP 支持 | 理由 |
|------|----------|------|
| 中国大陆 | 否 | 虚拟币交易政策限制 |
| 美国 | 否 | SEC/CFTC 监管复杂 |
| 香港 | 是 | VASP 牌照框架清晰 |
| 新加坡 | 是 | MAS 监管友好 |
| 欧盟 | 是 | MiCA 框架逐步落地 |

### 合规检查清单
- [ ] KYC/AML 流程（MVP 内部使用可跳过）
- [ ] 风险披露声明
- [ ] 交易记录保存（≥5年）
- [ ] 审计日志完整性
